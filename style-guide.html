F · Segmented Control

2–4 options. Active button gets class .on via JS (toggle aria-pressed too for accessibility). Never more than 4 options — use a <select> beyond that.

G · Simple / Advanced Toggle

The .adv-bar is always the FIRST child of the controls column — never above the grid. Any element with class .adv is hidden until body has class .adv-on. Click the buttons below to try it.
